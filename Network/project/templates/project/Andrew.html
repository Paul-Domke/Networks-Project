Recent DDoS Attacks

Paul Andrew Domke

The 2016 Dyn attack

The 2016 Dyn attack was a DDOS attack on the domain name service provider Dyn. This attack was accomplished with tens of millions of ip addresses targeting the systems operated by Dyn. This attack largely affected internet services and platforms in regions of North America and Europe. The sites that were affected include Twitter, Amazon, GitHub, and The New York Times. The groups Anonymous and New World Hackers took responsibility for this attack. The attack was made when a bot-net of tens of millions of ip addresses that belonged to internet connected devices such as; printers, baby cameras, ip cameras, residential gateways, etc. that were infected with Mirari malware made an incredible amount of DNS lookup requests. The Mirari source code was posted online after the attack, it included the default credentials of over 60 devices which enabled the perpetrators to hack many devices. On December 13 2017, the Justice Department announced that the three men, Paras Jha, Josiah White, and Dalton Norman, plead guilty to crimes relating the the cyber attack on Dyn.

The 2015 GitHub attack

The 2015 GitHub attack was a DDoS attack that is believed to be politically motivated. Although the attack was on GitHub two specific urls were targeted. Each of these urls belonged to GitHub repositories whose goals were to thwart Chinese state censorship. The attack originated from devices that visited Baidu, China’s number one search engine. Sites that used Baidu’s analytics services were also part of this attack. They were injecting JavaScript code that would send HTTP requests to the two GitHub pages. It was later found that the malicious code did not come from Baidu. This code was added by and intermediary service.

The 2013 Spamhaus attack

Spamhaus is service that tracks email spammers and spammer related activity. The attack on Spamhaus was targeted initially towards its websites, mail servers, and nameservers. Over the next two weeks their supporting networks and services were attacked, this including a couple of internet exchanges. Two arrest were made following this attack, one being a Dutch national living in Spain, the other A 17 year old male from London.
